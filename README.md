# 3D Printable Logo

## Overview

This project is a custom 3D printable logo designed in Onshape.

**Onshape Model:**
https://cad.onshape.com/documents/37c45f967a1bc340a5eedead/w/e0af9ace00915cc2a413577c/e/b9eca3e0131fe57171ddffc4

## Design

* Created using **Spline** sketches for smooth curves.
* Extruded into a 3D model.
* Designed for easy single-color 3D printing.

* Onshape
